# CodeAlpha To-Do List Application

## Description
This is a simple console-based To-Do List Application developed in C++.

## Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Save tasks to a file
- Load tasks from a file

## Technologies Used
- C++
- File Handling
- Object-Oriented Programming

## How to Run
1. Compile the program:
   g++ todo.cpp -o todo
2. Run the executable:
   todo.exe
